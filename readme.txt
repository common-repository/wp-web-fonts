== Description ==

Brighten up your blog with Web fonts from google. Take a look at the <a href="http://wordpress.org/extend/plugins/wp-web-fonts/screenshots/">screenshots area</a> area to give you an idea how easy it is.
Search hundreds of web font families for example on http://www.google.com/webfonts, then add the one you like to your Blog!

== Installation ==

This section describes how to install the plugin and get it working.

   1. Upload wp-web-fonts.php to the /wp-content/plugins/wp-web-fonts/ directory.
   2. Activate the plugin through the ‘Plugins’ menu in WordPress
